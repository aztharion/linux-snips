# Linux Snips

A collection of quick installation scripts for popular applications on Linux (Debian/Ubuntu).

## Purpose

This repository contains ready-to-use bash scripts that allow you to quickly install and configure applications on Linux systems. Simply copy the entire code block and paste it into your terminal.

## Available Applications

- [Postman](postman.md) - API development and testing tool

## Usage

1. Navigate to the application guide you want to install
2. Copy the entire bash script block
3. Paste it into your terminal
4. Press Enter and wait for the installation to complete

## Requirements

- Debian/Ubuntu-based Linux distribution
- `sudo` privileges
- Internet connection

## Contributing

Feel free to add more application installation scripts by creating a pull request. Please follow the existing format:
- One markdown file per application
- Clear instructions
- Single copy-paste script block
- Include cleanup of previous installations if applicable
